# eID Easy js SDk
eideasy-js-sdk provides you with a simple set of functions to get the user's identity.

## Installing

Using jsDelivr CDN:

```html
<script src="https://cdn.jsdelivr.net/npm/@eid-easy/eideasy-js-sdk/dist/eideasy-js-sdk.min.js"></script>
```
